Raspberry Pi Dashboard

This is an example of a real-time dashboard running on Google App Engine with Raspberry Pi Zero sending it sensory data using Cloud Endpoints. The dashboard uses Google Charts for visualization and the Google App Engine Channel API for real-time updates. The voltage data is from my 6.5 x 6.5cm solar cell. More details here
